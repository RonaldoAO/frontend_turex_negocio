## 📌 Visión General
**TUREX** es la infraestructura digital diseñada para capturar y redistribuir la derrama económica del Mundial 2026 hacia el corazón cultural de México. Este repositorio contiene la aplicación móvil desarrollada en **Flutter**, enfocada en transformar el viaje del fan mundialista en una búsqueda de autenticidad y legado cultural.

## 🚀 Características Principales
* **Soft-planning Inteligente:** Itinerarios dinámicos que conectan las sedes mundialistas (CDMX, GDL, MTY) con el nodo cultural de Oaxaca, optimizando los 13 días de estancia promedio del turista.
* **Sistema de Gamificación:** Misiones culturales que recompensan el consumo en negocios locales (Artesanía, Gastronomía e Inmersión) con activos digitales y accesos exclusivos a la Guelaguetza.
* **Mascota IA Interactiva:** Un asistente bilingüe que explica el contexto histórico y artesanal de cada sitio mediante realidad aumentada y procesamiento de lenguaje natural.
* **Interfaz de Usuario de Alta Fidelidad:** Diseño inspirado en la iconografía oaxaqueña con estándares de usabilidad internacional.

## 🛠️ Stack Tecnológico
* **Framework:** Flutter (Dart)
* **Gestión de Estado:** Bloc / Provider
* **Comunicación:** REST API con el backend de TUREX
* **Mapas:** Google Maps API (Custom Styles)
